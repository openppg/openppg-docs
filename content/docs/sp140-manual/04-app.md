## App Features

The OpenPPG app provides essential functionality for your SP140 v2.5:

### Battery Monitoring
- Real-time cell voltages
- State of Charge (SOC) display
- Temperature monitoring
- Charge cycle history
- Health diagnostics

### System Configuration
- Motor controller settings
- Throttle response curves
- Power limiting options
- Alert thresholds
- Cruise control settings

### Flight Logging
- Automatic flight detection
- GPS track recording
- Flight statistics
- Battery usage per flight
- Hour meter tracking

### Diagnostics
- System error codes
- Component status
- Firmware versions
- Connection status
- Maintenance reminders

## Installation

### iOS (iPhone/iPad)
1. Scan the QR code or visit the App Store
2. Search for "OpenPPG"
3. Download and install
4. Allow Bluetooth permissions when prompted

### Android
1. Scan the QR code or visit Google Play
2. Search for "OpenPPG"
3. Download and install
4. Allow Bluetooth and location permissions when prompted

## Connecting to Your SP140

1. **Turn on your SP140**: Flip the main power switch on the battery
2. **Open the OpenPPG app**
3. **Enable Bluetooth**: Make sure Bluetooth is enabled on your phone
4. **Tap Connect**: The app will automatically search for your SP140
5. **Select Your Device**: Choose your SP140 from the list
6. **Wait for Connection**: Connection typically takes 5-10 seconds

{{< callout context="tip" title="Connection Tip" icon="outline/rocket" >}}
Keep your phone within 10 meters (30 feet) of the SP140 for best connection reliability.
{{< /callout >}}

## Using the App

### Main Dashboard

The main screen shows:
- Current battery SOC
- System voltage
- Cell voltages (tap to expand)
- Temperature readings
- Flight hours
- Last flight date

### Battery Health Check

Before each flight:

1. Open the app
2. Connect to your SP140
3. Check battery status:
   - All cells should be balanced (within 0.05V)
   - Temperature should be moderate
   - No error codes present
   - SOC appropriate for planned flight

### System Configuration

{{< callout context="caution" title="Advanced Users Only" icon="outline/alert-triangle" >}}
Changing system settings can affect performance and safety. Only adjust settings if you understand the implications.
{{< /callout >}}

To access settings:
1. Connect to SP140
2. Tap the settings icon
3. Review current configuration
4. Make changes as needed
5. Confirm and save

### Flight Logs

View your flight history:
1. Tap "Flight Logs"
2. Browse flights by date
3. Tap a flight for details:
   - Duration
   - Distance
   - Max altitude
   - Battery used
   - GPS track (if available)

### Firmware Updates

Keep your system up to date:

1. Check for updates in the app
2. Follow on-screen instructions
3. Ensure battery is > 50% SOC
4. Do not disconnect during update
5. Restart system after completion

## Troubleshooting

### Cannot Connect
- Verify Bluetooth is enabled
- Ensure SP140 is powered on
- Close and reopen the app
- Restart your phone
- Check for app updates

### Connection Drops
- Keep phone within range
- Avoid interference from other devices
- Ensure phone battery is charged
- Check for app updates

### Data Not Updating
- Disconnect and reconnect
- Restart the app
- Power cycle the SP140
- Check for firmware updates
