Welcome to your OpenPPG SP140 v2.5! This guide will help you get started quickly.

## Quick Start Steps

1. **Watch the Video Tutorials**
   - Visit [openppg.com/sp140/v2.5/video](https://openppg.com/sp140/v2.5/video)
   - Watch frame assembly and power pack videos

2. **Download the OpenPPG App**
   - Essential for battery monitoring and system configuration
   - See [App Guide](/docs/sp140-manual/04-app/)

3. **Read the Complete Manual**
   - [SP140 v2.5 User Manual](/docs/sp140-manual/)
   - Start with [Technical Specifications](/docs/sp140-manual/01-specifications/)

4. **Assembly**
   - Follow the [Assembly Guide](/docs/sp140-manual/assembly/)
   - Time required: 30-60 minutes

5. **First Flight Preparation**
   - Complete [Pre-Flight Checks](/docs/sp140-manual/07-pre-flight/)
   - Review [Flight Operations](/docs/sp140-manual/08-flight-ops/)
